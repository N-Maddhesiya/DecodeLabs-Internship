# Movie Recommendation System

This is a simple AI-style recommendation system based on user preferences and similarity scoring.

## Features
- Takes user preferences as input
- Matches preferences with movie attributes
- Calculates a similarity score
- Displays top movie recommendations
- Gives reasons for each recommendation
