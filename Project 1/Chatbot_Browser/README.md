# Rule-Based AI Chatbot

A simple rule-based AI chatbot built to respond to user inputs using predefined patterns and responses.

## Features

* Rule-based conversation flow
* Easy to customize responses
* Lightweight and beginner-friendly
* Works as a basic AI chatbot demo project

## Tech Stack

* HTML
* CSS
* JavaScript
* Python

## How It Works

1. The user enters a message.
2. The chatbot checks the input against predefined rules.
3. A matching response is returned.
4. If no match is found, a default reply is shown.
